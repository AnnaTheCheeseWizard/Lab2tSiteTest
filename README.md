# Lab2tSiteTest
Roll Club UI Automation Suite

This project is a dedicated automated testing suite designed to ensure the reliability and performance of the Roll Club web interface. Developed as a university assignment, it focuses on verifying critical user flows, such as menu navigation, product filtering, and shopping cart accessibility, to ensure a seamless ordering experience.

The automation is built using Python 3.14, the Pytest framework, and Selenium WebDriver for browser interaction. To achieve high stability, the suite implements robust element-locating strategies and asynchronous wait mechanisms, effectively handling dynamic content across different sections of the website including rolls, promotions, and checkout pages.
